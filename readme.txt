Vinyl is a audio player plugin.

== Description ==

The Vinyl plugin allows you to embed audio files and playlists in your website. It is
a simple and easy to use plugin that allows you to add audio files to your website
using the WordPress block editor.

The plugin is still in development and we are working on adding more features and
improving the user experience.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/vinyl` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress

= Minimum Requirements =

* WordPress 6.3 or greater
* PHP version 7.4 or greater

== Source Code ==

The source code for this plugin is available on [GitHub](https://github.com/wp-blocks/vinyl).

This plugin is uses the library [media-chrome](https://github.com/muxinc/media-chrome) to render
the audio player.
